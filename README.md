# 🐦 Twitter Sentiment Analysis using Machine Learning & NLP

## 📌 Project Overview

This project performs sentiment analysis on Twitter posts using Natural Language Processing (NLP) and Machine Learning techniques. The model classifies tweets into positive and negative sentiments by applying text preprocessing, feature extraction, and supervised machine learning algorithms.

The objective is to understand public opinion from social media data, which can be useful for businesses, product reviews, customer feedback analysis, and market research.

---

## 🚀 Features

- Text preprocessing using NLP techniques
- Data cleaning and normalization
- Stopword removal
- Stemming
- TF-IDF Vectorization
- Machine Learning model training
- Sentiment prediction for tweets
- Model performance evaluation
- Easy-to-use prediction notebook

---

## 📂 Dataset

**Dataset:** Sentiment140 Dataset

- 1.6 Million Twitter Tweets
- Binary Sentiment Classification
- Labels:
  - 0 → Negative
  - 4 → Positive

Dataset Source:
https://www.kaggle.com/datasets/kazanova/sentiment140

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Twitter-Sentiment-Analysis.git
```

Move into the project directory

```bash
cd Twitter-Sentiment-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

## ▶️ Run the Project

Open the notebook

```bash

```

or upload the notebook to **Google Colab** and run all cells.
